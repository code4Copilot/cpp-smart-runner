## 🐛 修正的問題

### Big5/GBK 編碼偵測誤判
- **問題**：Big5 檔案被誤判為 GBK
- **影響**：雖然轉換功能正常，但顯示錯誤的原始編碼名稱
- **修正**：改進啟發式演算法，正確識別 Big5 特有特徵

## 💡 升級說明

### 如何更新

1. **編譯新版本**
   ```bash
   npm run compile
   ```

2. **重新載入 VS Code**
   - 按 `F1` → 選擇 `Reload Window`
   - 或關閉並重新開啟 VS Code

3. **驗證修正**
   - 開啟 Big5 檔案
   - 右鍵選擇「轉換為 UTF-8」
   - 檢查顯示訊息是否正確
