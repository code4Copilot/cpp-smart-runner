```

  │
  ▼
┌─────────────────┐

│ 或 F5 除錯      │
└─────────────────┘
  │
  ▼
┌─────────────────┐
│ runTest.ts      │  ← 測試執行器
│ - 設定環境      │
│ - 啟動 VS Code  │
└─────────────────┘
  │
  ▼
┌─────────────────┐
│ suite/index.ts  │  ← 測試套件入口
│ - 載入測試      │
│ - 配置 Mocha    │

  │
  ├────────────────┬────────────────┐
```
