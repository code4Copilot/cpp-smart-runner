# Version 1.1.1 Summary

**版本**: 1.1.1  
**發布日期**: 2026-01-29  
**類型**: Bug Fix Release

## 快速摘要

這是一個錯誤修正版本，解決了兩個重要的使用者體驗問題：
1. ✅ Big5 轉換前檢查未儲存修改，防止資料丟失
2. ✅ 自動編碼轉換改為可撤銷操作，行為更一致

## 修正的問題

| 問題 | 嚴重性 | 狀態 |
|------|--------|------|
| Big5 轉換未檢查未儲存修改 | 🟡 MEDIUM | ✅ 已修正 |
| 自動編碼轉換不可撤銷 | 🟢 LOW | ✅ 已修正 |

## 主要變更

### 1. Big5 轉換安全性改進
```typescript
// ✅ 修正後: 會檢查並警告使用者
if (document.isDirty) {
    const action = await vscode.window.showWarningMessage(
        '⚠️ 檔案有未儲存的修改，轉換為 Big5 前需要先儲存',
        '儲存並轉換',
        '取消'
    );
    // ...
}
```

### 2. 自動轉換可撤銷性改進
```typescript
// ✅ 修正後: 使用 editor.edit() 可撤銷
const success = await editor.edit(editBuilder => {
    editBuilder.replace(fullRange, result.content);
});

if (success) {
    await document.save(); // 自動儲存
}
```
