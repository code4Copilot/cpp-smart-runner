# 版本 1.1.0 快速參考

## 🚀 快速開始

### 如何使用新的子選單

```
1. 開啟任何 .c 或 .cpp 檔案
2. 在編輯器中按右鍵
3. 選擇「轉換編碼」
4. 選擇適合的轉換選項
5. 檢查結果
6. 按 Ctrl+Z 撤銷或 Ctrl+S 儲存
```

## 📋 子選單選項

| 選項 | 用途 | 何時使用 |
|------|------|----------|
| 轉換為 UTF-8 (自動偵測) | 自動判斷原始編碼 | 不確定檔案編碼時 |
| 從 Big5 轉換為 UTF-8 | 從 Big5 轉為 UTF-8 | 確定是 Big5 檔案 |
| 從 GBK 轉換為 UTF-8 | 從 GBK 轉為 UTF-8 | 確定是 GBK 檔案 |
| 轉換為 Big5 編碼 | 從 UTF-8 轉為 Big5 | 要在 Dev-C++ 開啟 |

## ⌨️ 快捷操作

| 操作 | 快捷鍵 | 說明 |
|------|--------|------|
| 撤銷轉換 | `Ctrl+Z` | 立即恢復原內容 |
| 儲存轉換 | `Ctrl+S` | 確認並寫入檔案 |
| 編譯程式 | `Ctrl+Alt+C` | 編譯當前檔案 |
| 執行程式 | `Ctrl+Alt+R` | 執行已編譯程式 |
| 編譯並執行 | `Ctrl+Alt+B` | 編譯後立即執行 |

## 🎯 常見場景

### 從 Dev-C++ 匯入

```
檔案：test.c (Big5 編碼，中文亂碼)
步驟：
1. 右鍵 → 轉換編碼 → 從 Big5 轉換為 UTF-8
2. 檢查中文顯示
3. Ctrl+S 儲存
```

### 不確定編碼

```
檔案：unknown.cpp (編碼不明)
步驟：
1. 右鍵 → 轉換編碼 → 轉換為 UTF-8 (自動偵測)
2. 如果錯誤 → Ctrl+Z 撤銷
3. 右鍵 → 轉換編碼 → 從 Big5/GBK 轉換
4. 找到正確的 → Ctrl+S 儲存
```

### 匯出到 Dev-C++

```
檔案：mycode.c (UTF-8 編碼)
步驟：
1. 右鍵 → 轉換編碼 → 轉換為 Big5 編碼
2. 檔案自動儲存
3. 在 Dev-C++ 中開啟
```

## 💡 提示

1. **轉換前先試自動偵測**，失敗再手動選擇
2. **善用 Ctrl+Z**，轉換錯誤立即撤銷
3. **Big5 和 GBK 容易混淆**，多試幾次
4. **轉為 Big5 會自動儲存**，無法撤銷
5. **儲存前先檢查**，確認中文正確顯示
