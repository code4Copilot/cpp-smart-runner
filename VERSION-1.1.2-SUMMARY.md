## 💡 使用提示

### Big5 轉換流程

1. **開啟 C/C++ 檔案**（UTF-8 編碼）
2. **右鍵 → 轉換編碼 → 轉換為 Big5**
3. **閱讀警告訊息**（重要！）
4. **點擊「確定轉換」**
5. **檔案自動關閉**
6. **用 Dev-C++ 開啟檔案**

### 轉回 UTF-8

1. **在 VS Code 重新開啟 Big5 檔案**（會顯示亂碼）
2. **右鍵 → 轉換編碼 → 從 Big5 轉換為 UTF-8**
3. **亂碼變回正常中文**
4. **可以繼續編輯**

## 🎓 適用場景

### ✅ 推薦使用 Big5 轉換
- 需要在 Dev-C++ 中開啟檔案
- 使用舊版編譯器（僅支援 Big5）
- 與他人協作（對方使用 Big5 環境）

### ✅ 推薦使用 UTF-8
- 在 VS Code 中開發
- 使用現代編譯器（支援 UTF-8）
- 跨平台開發

## ⚠️ 注意事項

1. Big5 轉換後檔案會**自動關閉**
2. VS Code 無法直接編輯 Big5 檔案
3. 需要使用 Dev-C++ 或其他 Big5 編輯器
4. 可以隨時轉回 UTF-8 繼續在 VS Code 編輯
